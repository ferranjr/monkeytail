Monkeytail
=========================

Validation utilities for cats.

This project is similar to other excellent projects like https://github.com/wix/accord and https://github.com/tobnee/DValidation 
but differs in that it focuses entirely on supporting cats.Validation.

Monkeytail offers macro based builders to quickl build instances of Validator[T]
